Welcome user to TestApp! This application will allow you to do the following:

  • View your Facebook profile
  • View your Twiiter profile
  • Update your Facebook status
  • Tweet a message to Twitter

You will need a Facebook or Twitter account to post and access your profile from these social media sites.

To know more about this application, please visit the tutorial at Spring Social Tutorial.

The GitHub repository is available at spring-social-tutorial.
